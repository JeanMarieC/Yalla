04Motion · responsive · handoff
Motion
  • Sheets & drawers slide on a spring (~280ms, ease-out); backdrop fades 0→.18 opacity.
  • Pins settle with a short drop-in; route line draws on with a dash offset.
  • Map fly-to on stop select; Pulse pins breathe on a 2.2s glow loop, staggered.
  • Presence avatars float subtly; "route updated" banner slides down once.
  • All gated behind prefers-reduced-motion — falls back to instant + cross-fade.
Responsive
  • Mobile-first; one breakpoint at md (768) for the split layouts.
  • Trip & finder: stacked map + sheet on mobile → side-by-side panel on desktop.
  • Stop card is a bottom sheet (grab handle) on mobile, right drawer ≥md.
  • Touch targets ≥44px; chips, toggles and pins all finger-sized.
  • Grids reflow 1→2→3 columns; type scale eases up ~15% at desktop.
Tokens → Tailwind
  • Colors as CSS vars (--paper, --ink, --terracotta, --sage) → theme.extend.colors.
  • Radii: 2xl/3xl; shadows: sm/md/lg as named tokens.
  • Newsreader (display) + Hanken Grotesk (sans) via next/font.
  • Each frame maps to one App-Router route; sheets/drawers are client components.
  • WCAG AA: ink on paper ≈ 13:1; terracotta text only on tint, never on white below 16px.
